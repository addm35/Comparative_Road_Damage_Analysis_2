# Comparative_Road_Damage_Analysis_2
Road maintenance is crucial for reducing risks caused by potholes. This work compares RT-DETR, Mask R-CNN, and YOLO11-seg for automated pavement damage detection using Computer Vision. The study evaluates accuracy, speed, and pothole area estimation to identify the most effective solution for automated road inspection.

To successfully run the code in the 'Comparative_Road_Damage_Analysis.ipynb' notebook, you will need to follow these steps:

1. Setup
Upload the folder named "colab_notebooks" (which contains the dataset and model files) directly to the root directory of your personal Google Drive (named My Drive).

2. Google Drive Authentication
When you execute the notebook cells, you will be need to authorize access to your Google Drive.


Note: Ensure you log into the same Google account where you uploaded the "colab_notebooks" folder to allow the notebook to access the necessary data.
